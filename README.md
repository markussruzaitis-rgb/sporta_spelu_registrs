# sporta_spelu_registrs
Dotā programma ļauj ievadīt sporta spēļu rezultātus (abu komandu nosaukumus, savstarpējās spēles rezultātu un spēles datumu).

Programmas darbības:
1. Pievieno spēli - abas spēlējošās komandas, rezultātu un datumu.
2. Parāda spēļu sarakstu.
3. Ļauj dzēst spēles no saraksta.
4. Automātiski kārto spēles pēc datuma, sākot no visaktuālākās.
5. Ļauj kārtot pēc alfabēta - no A līdz Z vai no Z līdz A.

Kā palaist:
1. Terminālī ierakstīt - pip insatall flask.
2. Terminālī ierakstīt - python app.py.
3. Atvērt http://127.0.0.1:5000.
